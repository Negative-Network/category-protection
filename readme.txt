**Category-Protection** Password protect your categories. Also work for pages if you add a page/category plugin
**Warning:** Uninstalling / deactivating this plugin will remove all the category protection

== Description ==

Protect pages/posts output with a password on a category level.
This plugin is not meant for a very high level protection.

== Installation ==

1. Go to your WordPress Admin
2. Go to Plugins -> Install Plugins
2. Search for Category Protection
3. Install

**Or:**

1. Download Category Protection
1. Unzip
1. upload the extracted directory to the /wp-content/plugins directory
1. activate
